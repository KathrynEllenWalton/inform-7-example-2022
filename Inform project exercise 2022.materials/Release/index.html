A work of interactive fiction.

Inform project exercise 2022 was created with Inform and has IFID 4811B238-1DB8-453B-9DA1-E5C8CD812C4A. To play a work like this one, you need an interpreter program: many are available, among them Gargoyle for Mac OS X or Windows; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
